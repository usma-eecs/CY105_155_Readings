The examples in the previous sections work fine but evaluate specific (static) examples that use the same numbers each time the program is run. What if we wanted to write a program to accept input from the user (the person actually running the code)?

In order to do this, we need a way to get input from the user. Luckily, in Python there is a built-in function to accomplish this task. As you might expect, it is the input() function. See below for an example from your reference card of prompting the user to enter their name and then storing the name the enter into a variable name: 

user_name = input("Please enter your name: ")

Review the print() and input() explanations and example from page 1 of your reference card:

Built-in Functions
print(item) Displays the argument item on the screen. Does not return anything.
input('prompt') Prompts user for input with the text prompt. Note that this always returns a string, so you might have to convert it to something more useful with the int() or float() funcô€†Ÿons. 
# Input and Print
name = input('What is your name? ')
whole_num = int(float(input('Guess a number: ')))
print(name + ' your number was: ' + str(whole_num))
dec_num = float(input('Guess a number: '))
print('Your decimal number was: ' + str(dec_num))

The input function allows the user to provide a prompt string. When the function is evaluated, the prompt is shown. The user of the program can enter the name and press return. When this happens the text that has been entered is returned from the input function, and in this case assigned to the variable name. Make sure you run this example a number of times and try some different names and numbers as the input.

It is very important to note that the input() function returns a string value. Even if you asked the user to enter their age, you would get back a string like "17". It would be your job, as the programmer, to convert that string into an integer or a floating point number, using the int() or float() converter functions we saw earlier.
