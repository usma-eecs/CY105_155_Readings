The color methods in Python use either a color string or a tuple with red, green, and blue values. All valid Hypertext Markup Language (HTML) color strings can be utilized. The RGB colors below represent a percent of red, green, or blue intensities.

There are three colors options:

  1. Color Name -- "ROSYBROWN"
  2. Hex Code -- "#BC8F8F"
  3. RGB -- 0.74, 0.56, 0.56

The color string or tuple can be used as arguments with the .bgcolor(), .fillcolor(), and .pencolor() functions.
