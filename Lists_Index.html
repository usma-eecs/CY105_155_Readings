The indexing operator (Python uses square brackets to enclose the index) selects a single item from a list. The items are accessed by their position or index value. For example, in the list shown below, the 10 characters are indexed left to right from position 0 to position 9.

It is also the case that the positions are named from right to left using negative numbers where -1 is the rightmost index and so on. Note that the character at index 4 (or -6) is the blank character.

The expression school[2] selects the character at index 2 from school, stores the single character in a char.

Remember that computer scientists often start counting from zero. The letter at index zero of ["W", "e", "s", "t", " ", "P", "o", "i", "n","t"] W. So at position [2] we have the letter s.

If you want the zero-eth item of a list, you just put 0, or any expression with the value 0, in the brackets. Give it a try.

The expression in brackets is called an index. An index specifies a member of an ordered collection. In this case the collection of characters in the list. The index indicates which character you want. It can be any integer expression so long as it evaluates to a valid index value.
